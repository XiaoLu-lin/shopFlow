## 2. 订单评价发布页补齐

- [x] 新增 `shopflow-h5-next` 订单评价发布页。
- [x] 接通 `order/goods`、`order/comment` 与 `storage/upload`。
- [x] 实现评分、评价内容、图片上传与提交流程。

验收契约：

- 已具备评价资格的订单商品 MUST 能进入评价页并查看待评价商品信息。
- 评价提交成功后 SHALL 给出明确反馈并进入可预期的后续页面。
- 评论失败时 MUST 给出可见错误提示，不得停留为空白或静默失败。

## 3. 订单页评价入口补齐

- [x] 在订单列表中补齐“去评价”入口与已评价状态表现。
- [x] 在订单详情中补齐“去评价”入口与已评价状态表现。
- [x] 确保仅在后端返回允许评价时展示入口。

验收契约：

- 可评价订单 MUST 能从订单列表进入评价页。
- 可评价订单 MUST 能从订单详情进入评价页。
- 不可评价订单不得展示误导性“去评价”入口。
