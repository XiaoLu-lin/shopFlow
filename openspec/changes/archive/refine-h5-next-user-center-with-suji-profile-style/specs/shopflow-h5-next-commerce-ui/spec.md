# shopflow-h5-next 商城 UI 规格

## MODIFIED Requirements

### Requirement: 个人中心必须支持更贴近 Suji Profile A 方案的用户主页结构

`shopflow-h5-next` 的个人中心页 MUST 在保持 `ShopFlow` 品牌和现有服务入口能力的前提下，采用更贴近用户确认的 Suji Profile A 方案结构。

#### Scenario: 个人中心展示三段式用户主页结构

- **前提** 用户进入个人中心页
- **当** 页面完成渲染
- **则** 页面必须展示渐变用户头部、订单状态五宫格和列表式服务入口三段结构
- **且** 不得继续以双列说明卡作为主要服务入口形态

#### Scenario: 个人中心保留现有入口与真实数据边界

- **前提** 用户浏览个人中心并点击入口
- **当** 页面渲染和处理跳转
- **则** 页面必须继续提供订单、购物车、收藏、地址、优惠券、售后、帮助、服务和设置等现有入口
- **且** 头部文案只能消费现有昵称、头像、手机号等真实字段
- **且** 不得伪造用户 ID、会员等级、积分等当前不存在的数据
